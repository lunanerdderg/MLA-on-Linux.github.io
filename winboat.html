How to use Winboat

Winboat will NOT work if you have less than 4 GiB of RAM and 2 CPU cores.

  1. Download your version of Winboat from this site and install it.
  2. Continue through the slides until you see one called "Pre-Requisites". Check the bottom to see if FreeRDP is installed. If not, install the Flatpak by running these commands.
  3. Check if you have Docker installed. If you do not, I recommend switching the dropdown to Podman because it is easier to install and use. If you prefer to use Docker, follow the instructions at the links Winboat provides you.
  4. How to install Podman Compose:
    1. Install your distribution's version of Podman from here. If not listed, get the Flatpak.
    2. Use pip to install Compose:
      pip3 install podman-compose
      If that doesn't work, try installing using your package manager, or manually, according to these instructions.
  5. Continue through the next few screens, following Winboat's instructions, and wait for Windows to install. (It's going to take a long time!)
  6. Once Windows is finished installing, stop the virtual machine with the big red square button, then hit the play button when it's done. (This will restart Windows, which can prevent some common errors.)
  7. Wait until the Winboat API is Online, then switch to the "Apps" tab and open Microsoft Edge, (or open Windows Store and install your preferred browser).
  8. Copy and paste this link into the browser:
    https://portal.office.com/account
  9. Sign in with your Microsoft account, then switch to the "Apps & devices" tab and click the "Install Office" button to download the installer.
  10. Once it's finished downloading, close Edge and open File Explorer. Find the installer in your Downloads folder and run it.
  11. If you're using a school account: Go through all the steps until you reach the "Stay signed in to all your apps" page. Unselect the "Allow my organization to manage my device" option because this causes problems with virtual machines. Then press "Ok".
  12. Once the installer is finished, close File Explorer and restart the virtual machine again (step 6).
  13. Now Microsoft Word should be one of the apps on your "Apps" tab! Since this is essentially an entire Windows installation running through a virtual machine, you can install Onedrive on it to sync your files if you wish.
