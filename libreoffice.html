Word Online LibreOffice Writer Winboat
  1. Plenty of Linux distributions have LibreOffice already installed, so check if you already have it before trying to install it.
  2. If you don't have it, install the version of LibreOffice for your distribution:
    Distribution Install link
    Debian-based distributions:
    • Linux Mint
    • ZorinOS
    • Ubuntu
    		 DEB package
    Red Hat-based distributions:
    • Fedora
    • CentOS
    • Bazzite
    		 RPM package
    Other distributions:
    • Flatpak
    • Snap
    • AppImage

  3. Install the Times New Roman font.
    • Debian-based distribution users, copy and paste this into your terminal:

      sudo apt-get update && sudo apt-get install ttf-mscorefonts-installer -y

    • Arch-based distribution users, copy and paste this into your terminal:

      if ! pacman -Qi base-devel > /dev/null ; then
      sudo pacman -S base-devel
      fi
      git clone https://aur.archlinux.org/ttf-ms-win11-auto.git
      cd ttf-ms-win11-auto/
      mkpkg -si
      rm -rf ttf-ms-win11-auto
