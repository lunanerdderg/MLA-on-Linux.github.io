How to use LibreOffice on Linux

  1. Plenty of Linux distributions have LibreOffice already installed, so check if you already have it before trying to install it.
  2. If you don't have it, install the version of LibreOffice for your distribution:
    Distribution Install link
    Debian-based distributions:
    • Linux Mint
    • ZorinOS
    • Ubuntu
    		 DEB package
    Fedora-based distributions:
    • CentOS
    • Bazzite
    • Nobara
    		 RPM package
    Other distributions:
    • Flatpak
    • Snap
    • AppImage

  3. Install the Times New Roman font.
    • Debian-based distribution users, copy and paste this into your terminal:
      sudo apt-get update && sudo apt-get install ttf-mscorefonts-installer -y
    • Arch-based distribution users, copy and paste this into your terminal: (SteamOS, CachyOS, Manjaro)
      if ! pacman -Qi base-devel > /dev/null ; then
      sudo pacman -S base-devel
      fi
      git clone https://aur.archlinux.org/ttf-ms-win11-auto.git
      cd ttf-ms-win11-auto/
      mkpkg -si
      rm -rf ttf-ms-win11-auto
    • Fedora-based distribution users, copy and paste this into your terminal:
      sudo dnf install ttf-mscorefonts-installer -y
  4. Change default Print and Page settings:
    • By default, LibreOffice uses the international A4 style (8.3 x 11.7 ins). However, the United States of America uses US Letter (8.5 x 11 ins)
      Size comparison:
    • Changing default Print settings:
      1. Go into File --> Printer Settings and make sure your printer is selected, then go into Properties.
      2. In the "Paper Size" dropdown, select "US Letter", then hit "Ok".
    • Changing default Page settings:
      1. Go into Format --> Page Style and change the value of the "Format" dropdown to "Letter".
      2. Change all the "Margin" values to 1 (except for the Gutter), then hit "Apply".
